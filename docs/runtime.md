## Supported languages

Nimbella supports the following languages and runtimes for creating serverless APIs.
- .NET 3.1 
- Deno 1.3
- Go 1.5
- Java 8
- JavaScript and Node.js 14
- PHP 8.0
- Python 3.7
- Swift 4.2
- Ruby 2.6
- TypeScript 3.7
